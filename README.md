# Matrix

## Installing project:
1. Clone this repository to your computer:
```
git@github.com:Ignitor21/Matrix.git
cd Matrix
```
2. Run cmake:
```
cmake -B build
cd build 
cmake --build .
```

## Tests
To start testing move to `tests` directory and run:
```
./test.sh
```

## Unit tests
To start unit tests move to `build/unit-tests` directory and run:

```
./unit-tests
```
